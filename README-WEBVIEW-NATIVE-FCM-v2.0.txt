MK Movies Android Build Pack v2.0 - Native WebView + Native Firebase Notifications

This version is no longer a TWA/Chrome Custom Tab build. It uses a native Android WebView container, so it will not show the Chrome/TWA "running in Chrome" message.

Included:
- Native WebView app container for https://mkmovies.com.ng
- Native Firebase Cloud Messaging notifications
- Android 13+ notification permission request
- Native token registration to WordPress REST endpoint
- Pull-to-refresh
- Back button navigation
- File chooser support for upload fields
- DownloadManager support

Important:
- Keep the same package name: ng.com.mkmovies.app
- Keep the same GitHub secrets: MK_KEYSTORE_PASSWORD and MK_KEY_PASSWORD
- Install MK Movies Push Notifications Plugin v3.0 or newer in WordPress and keep Web/Chrome subscription OFF.
